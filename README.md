# Pokemon Search Engine (Pokedex)

## Overview
A full-stack Pokemon search engine that allows users to search for Pokemon and view detailed attributes.

## Tech Stack
- Backend: Node.js, Express, Axios, Node-Cache
- Frontend: HTML, CSS, JavaScript
- External API: PokeAPI

## Features
- Search Pokemon by name
- Random Pokemon search
- Caching for faster repeat queries
- REST-compliant APIs
- Clean and responsive UI
